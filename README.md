# Impano Initiative Funds

**Nourishing Futures, Empowering Children**

Welcome to the official website of Impano Initiative Funds. We are a movement of passionate young changemakers dedicated to helping children from underprivileged backgrounds receive food at school, study in safety, and build a brighter future.

## 🌟 Our Purpose

*   **Vision:** To create a future where no student goes hungry in school, and every child has the nourishment needed to reach their full potential.
*   **Mission:** To empower students by providing nutritious meals, safe education spaces, and opportunities for a brighter future.

## 🚀 Key Features

*   **Premium UI/UX:** Built with Next.js and Tailwind CSS 4, featuring modern glassmorphism, fluid animations, and a rich dark aesthetic.
*   **Dynamic Scroll Effects:** Intersection observers natively trigger CSS fade, reveal, and slide animations across sections.
*   **Highly Responsive:** Delivers an optimized experience on any device, ensuring broad accessibility.
*   **Brand Driven:** Focuses on the "Outfit" typography stack and custom `#f09620` brand coloring.

## 🛠 Tech Stack

*   **Framework:** [Next.js](https://nextjs.org/)
*   **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) & Vanilla CSS Modules
*   **Language:** [TypeScript](https://www.typescriptlang.org/)

## 📦 Getting Started

First, install the necessary dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
